=== Plugin Name ===
Tags: subtitle,persian,wordpress parsi,subtitle finder
== Description ==
by this plugin u can make a online subtitle Finder.
u can make a search form by [subtitle_finder_searchform] Shortcode.
and u can list subtitle of a plugin by [subtitle_finder movie='test'] Shortcoe in your posts and pages ( and subtitle_finder($name) function).
== Installation ==
1 . Upload plugin 
2 . Active plugin from plugins menu
3 . go to settings and Subtitle Finder and set your language and others
